Proposal

UofT SCS Data Analytics Team 6 Members

Sara Hovakeemian, Laurel Lobo, Jose Tomines, Jesse Gray

Project Scope

Analyze the distribution of Data Science positions (Data Analyst, Data Scientist, Data Engineer) in Canada by scraping Indeed. We will scrap Glassdoor to get salaries for these positions by position. We will also explore whether these positions are in cities/provinces that are appealing (cost of living) using the Numbeo API.

Breakdown of Questions

  1. How many positions are there per city/province
  2. What is the comparison between salary and cost of living

Potential Data Sources

  1. Indeed
  2. Glassdoor
  3. Numbeo API
  4. Stats Canada API
