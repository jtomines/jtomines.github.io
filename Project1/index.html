Are We Angry For Water?

Exploring how climate change impacts the global freshwater supply, and if low freshwater areas experience more conflict. The water status of the United States is assessed to see if there are any trends.

This is the first group project assignment for the University of Toronto Continuing School of Studies Data Analytics Boot Camp Inaugural Class. It is the first demonstration of gathering data from various public websites (via API or file download), using Python with Pandas & Matplotlib to clean and merge the data, and create the graphs to view data correlation and trends. The project was graded based on the presentation by the group which included a PowerPoint display. This website provides you with both the PowerPoint slides and a video of the presentation itself.
